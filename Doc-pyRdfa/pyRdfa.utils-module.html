quote_URI(uri, options=None)

source code 

'quote' a URI, ie, exchange special characters for their '%..' equivalents. Some of the characters may stay as they are (listed in _unquotedChars. If one of the characters listed in _warnChars is also in the uri, an extra warning is also generated.

Parameters:
  • uri - URI
  • options (Options)

traverse_tree(node, func)

source code 

Traverse the whole element tree, and perform the function func on all the elements.

Parameters:
  • node - DOM element node
  • func - function to be called on the node. Input parameter is a DOM Element Node. If the function returns a boolean True, the recursion is stopped.

return_XML(state, inode, base=True, xmlns=True)

source code 

Get (recursively) the XML Literal content of a DOM Element Node. (Most of the processing is done via a node.toxml call of the xml minidom implementation.)

Parameters:
  • inode - DOM Node
  • state - pyRdfa.state.ExecutionContext
  • base (Boolean) - whether the base element should be added to the output
  • xmlns (Boolean) - whether the namespace declarations should be repeated in the generated node
Returns:
string
