Wrapper around RDFLib's Graph object. The issue is that the serializers in RDFLib are buggy:-(

In RDFLib 2.X both the Turtle and the RDF/XML serializations have issues (bugs and ugly output). In RDFLib 3.X the Turtle serialization seems to be fine, but the RDF/XML has problems:-(

This wrapper provides a subclass of RDFLib’s Graph overriding the serialize method to register, if necessary, a different serializer and use that one.
