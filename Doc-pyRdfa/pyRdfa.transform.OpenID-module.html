Module OpenID

source code

Simple transfomer: handle OpenID elements. Ie: an openid namespace is added and the usual 'link' elements for openid are exchanged against a namespaced version.

Requires: RDFLib package

OpenID_transform(html, options, state)

source code 

Replace openid.XXX type @rel attribute values in <link> elements by openid:XXX. The openid URI is also added to the top level namespaces with the openid: local name.

Parameters:
  • html - a DOM node for the top level html element
  • options (Options) - invocation options
  • state (State) - top level execution state
