Class Hierarchy

  • pyRdfa.rdfs.cache.CachedVocabIndex: Class to manage the cache index.
    • pyRdfa.rdfs.cache.CachedVocab: Cache for a specific vocab.
  • pyRdfa.state.ExecutionContext: State at a specific node, including the current set of namespaces in the RDFLib sense, current language, the base, vocabularies, etc.
  • pyRdfa.host.HostLanguage: An enumeration style class: recognized host language types for this processor of RDFa.
  • pyRdfa.termorcurie.InitialContext: Get the initial context values.
  • pyRdfa.state.ListStructure: Special class to handle the @inlist type structures in RDFa 1.1; stores the "origin", i.e, where the list will be attached to, and the mappings as defined in the spec.
  • pyRdfa.host.MediaTypes: An enumeration style class: some common media types (better have them at one place to avoid misstyping...)
  • pyRdfa.rdfs.process.MiniOWL: Class implementing the simple OWL RL Reasoning required by RDFa in managing vocabulary files.
  • pyRdfa.options.Options: Settable options.
  • pyRdfa.property.ProcessProperty: Generate the value for @property taking into account datatype, etc.
  • pyRdfa.options.ProcessorGraph: Wrapper around the 'processor graph', ie, the (RDF) Graph containing the warnings, error messages, and informational messages.
  • pyRdfa.termorcurie.TermOrCurie: Wrapper around vocabulary management, ie, mapping a term to a URI, as well as a CURIE to a URI.
  • pyRdfa.utils.URIOpener: A wrapper around the urllib2 method to open a resource.
  • pyRdfa.initialcontext.Wrapper
  • object: The most base type
    • exceptions.BaseException: Common base class for all exceptions
      • exceptions.Exception: Common base class for all non-exit exceptions.
        • pyRdfa.RDFaError: Superclass exceptions representing error conditions defined by the RDFa 1.1 specification.
          • pyRdfa.FailedSource: Raised when the original source cannot be accessed.
          • pyRdfa.HTTPError: Raised when HTTP problems are detected.
          • pyRdfa.ProcessingError: Error found during processing.
        • pyRdfa.pyRdfaError: Superclass exceptions representing error conditions outside the RDFa 1.1 specification.
    • rdflib.term.Node: A Node in the Graph.
      • rdflib.graph.Graph: An RDF Graph
        • pyRdfa.graph.MyGraph: Wrapper around RDFLib's Graph object.
  • pyRdfa.pyRdfa: Main processing class for the distiller
