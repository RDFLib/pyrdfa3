Class InitialContext

source code

Get the initial context values. In most cases this class has an empty content, except for the top level (in case of RDFa 1.1). Each TermOrCurie class has one instance of this class. It provides initial mappings for terms, namespace prefixes, etc, that the top level TermOrCurie instance uses for its own initialization.
