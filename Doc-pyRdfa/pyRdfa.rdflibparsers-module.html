Module rdflibparsers

source code

Extraction parsers for structured data embedded into HTML or XML files. The former may include RDFa or microdata. The syntax and the extraction procedures are based on:

* The RDFa specifications: http://www.w3.org/TR/#tr_RDFa * The microdata specification: http://www.w3.org/TR/microdata/ * The specification of the microdata to RDF conversion: http://www.w3.org/TR/microdata-rdf/
